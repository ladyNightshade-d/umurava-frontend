# 🚀 Quick Start - HireWise Next.js

## Run the Application

### Option 1: Use the Batch File (Easiest)
```bash
run-nextjs.bat
```

### Option 2: Manual Commands
```bash
npm run dev
```

Then open: **http://localhost:3000**

## ✅ What You Have

- **Next.js 16** with App Router (as required by Umurava)
- **TypeScript** throughout
- **Redux Toolkit** for state management
- **Tailwind CSS** for styling
- **All original UI and functionality preserved**
